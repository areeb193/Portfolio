## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/areeb193/Portfolio.git
cd Portfolio
```

2. Install dependencies
```bash
npm install
```

3. Run the development server

```bash

5. Preview production build

```
